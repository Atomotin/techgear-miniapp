Импорт каталога в storage

Основной рабочий источник теперь storage и data/catalog.json. card-tovary.js оставлен только для legacy-импорта. Существующие записи не удаляются, совпадающие ID обновляются.

Каталог

Выбери карточку справа, чтобы быстро редактировать товар. Удаление выполняется сразу.

Категории

Категории управляют фильтрами в магазине и селектором в форме товара.

Заказы

Заказы, отправленные из Mini App через backend. Здесь можно менять статус обработки.

Клиенты

Список клиентов собирается из уже оформленных заказов. Здесь можно найти контакт, посмотреть количество заказов и быстро открыть Telegram или звонок.

Баннеры

Управление верхней каруселью Mini App: картинка, заголовок, кнопки и действия без правки кода.

Музыка
